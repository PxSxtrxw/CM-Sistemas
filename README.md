# CM-Sistemas
## App de Scraping en Python

### Lunes 24/6/24
Este es un código de prueba de scraping hecho en Python para la página [Cambios Chaco](https://www.cambioschaco.com.py) con el objetivo de recopilar los siguientes datos:

- Moneda
- Código
- Spread
- Compra
- Venta

### Librerías Usadas
- BeautifulSoup
- Requests
